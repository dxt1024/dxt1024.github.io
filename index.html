Hexo博客搭建心得

今天，照着网上的教程终于完成了Hexo博客的搭建并部署到了Github上。由于教程是在Mac OS上搭建的，而我的是Win10，所以有些许不同。现在就记录下来。

搭建准备

1.安装Git

直接从官网下载即可，运行安装程序，根据提示完成安装。
之后的命令行全在Git Bash下运行。

2.安装Node.js

也是从官网下载，推荐左边的长期支持的比较稳定的版本。下载后，运行安装程序，根据提示完成安装。
node -v验证是否安装成功。

3.安装Hexo

npm instal -g cnpm --registry=https://registry.npm.taobao.org安装淘宝npm镜像，加快后续文件的下载速度。如果对网速有自信，可以不安装，之后的cnpm命令全部改为npm。
安装Hexocnpm instal -g hexo-cli
验证是否安装成功heox -v
注意
1.npm的core-js版本可能会较低引发错误,需要更新版本cnpm install -g core-js@3

2.hexo 3以后server模块需要手动安装cnpm install -g hexo-server --save

开始搭建

1.创建工作目录

1.在合适的目录下创建新文件夹mkdir MyBlog2.初始化工作目录hexo init`

2.创建新博文

1.新建新博文hexo n "博文名"
2.在source/_posts/下找到“博文名.md”，并用Markdown语言编辑博文。

3.生成静态文件

hexo g

4.在本地浏览博客

1.开启Hexo服务heox s
2.在localhost:4000中浏览博客

部署到Github
